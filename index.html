api/AirEntries

Get all Air Entries from sensors

api/cities

Get all cities

api/AirEntries/cities/{city}

Get all air entries for the given city
Example:http://airmonit-2.apphb.com/api/AirEntries/cities/Coimbra

api/sensors

Get the sensors id and respectives cities

api/sensors/{id:int}

Get the air entries sent by specified sensor id
Example: http://airmonit-2.apphb.com/api/AirEntries/Sensors/1

api/ResumeAlarm/date/elements/{Element}

Get hourly summarized information(min, max and average values) for a parameter on a specific day in all cities
Example /api/ResumeAlarm/date/elements/O3?date=2017-11-19T08:15

api/ResumeAlarms/date/elements/{Element}/cities/{City}

Get hourly summarized information(min, max and average values) for a parameter on a specific day in a city specified by the user
Example api/ResumeAlarm/date/elements/O3/cities/Leiria?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08

api/ResumeAlarm/date/elements/{Element}/cities/{City}

Get daily summarized information(min, max and average values) for a parameter on a specific threshold(between two dates/time) in a city specified by the user or all cities;
Example api/ResumeAlarm/date/elements/O3/cities/Leiria?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08

api/Alarms/cities/{city}

Get all alarms on a given city
api/Alarms/cities/Leiria

api/Alarms/dates

Get all alarms between two dates
Example: api/Alarms/dates?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08

api/Alarms/dates/cities/{City}

Get Alarms resume between two dates on a given city
Example: api/Alarms/dates/cities/Leiria?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08

api/Alarms/dates/elements/{Element}/cities/{City}

Get all alarms on a given city for a given element between two dates
Example: api/Alarms/dates/elements/{Element}/cities/{City}?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08

api/userEntries

Get All User Entries

api/userEntries/cities/{city}

Get User Entries on a given city
Example: api/userEntries/cities/Leiria

api/userEntries/dates/cities/{City}

Get User Entries on a given city between two given dates
Example: api/userEntries/dates/cities/Leiria?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08

api/userEntries/dates

Get User Entries between two given dates
Example: api/userEntries/dates?dateIn=2017-11-18T23:08&dateOut=2017-11-22T23:08
